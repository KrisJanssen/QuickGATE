Quickgate

An easy general purpose tool for TCSPC based confocal imaging

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the QuickGate project webpage!

QuickGate is a general purpose tool that can be used to view and process confocal imaging data that was recorded using Time Correlated Single Photon Counting (TCSPC) equipment.

Currently, the software can handle following formats:

  1. PicoQuant TimeHarp 200 .t3r
  2. PicoQuant HydraHarp is COMING SOON!
